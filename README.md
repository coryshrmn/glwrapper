# GLWrapper

Header-only C++ wrapper for modern OpenGL and OpenGLES

## Features

* Full support for OpenGL 3.2 - 4.5 core profile, and OpenGLES 2.0 - 3.2

* Modern C++11 syntax, RAII with move support

* No heap allocations

* Header-only

* Packages OpenGL and OpenGLES headers

Barely started
